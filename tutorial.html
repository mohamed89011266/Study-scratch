Motion

1."Move 10 steps": This block makes the sprite move 10 steps in the direction it's facing.

2."Turn 15 degrees": This block makes the sprite turn 15 degrees clockwise.

3."Go to x: (0) y: (0)": This block moves the sprite to the specified coordinates (0, 0) on the stage.

4."Glide 2 secs to x: (0) y: (0)": This block smoothly moves the sprite to the specified coordinates over 2 seconds.

5."Point in direction (90)": This block changes the sprite's direction to face the specified angle (90 degrees in this case).

6."Point towards (Sprite2)": This block makes the sprite point towards another sprite called "Sprite2."

7.Change x by 5": This block adds 5 to the sprite's current x-coordinate, moving it horizontally.

8."Set x to 100": This block sets the sprite's x-coordinate to 100, moving it horizontally to that position.

9."If on edge, bounce": This block makes the sprite bounce when it reaches the edge of the stage.

10."Set rotation style [left-right v]": This block changes the rotation style of the sprite to either "left-right" or "don't rotate" (v stands for vertical).

looks

1."Say Hello! for 2 seconds": This block makes a speech bubble appear above the sprite, displaying the text "Hello!" for 2 seconds.

2."Show": This block shows the sprite if it was hidden.

3."Hide": This block hides the sprite from the stage.

4."Switch Costume to (Costume2)": This block changes the sprite's costume to "Costume2."

5."Next Costume": This block switches the sprite to the next costume in its costume list.

6."Change Size by 10%": This block increases or decreases the size of the sprite by 10%.

7."Go to Front Layer": This block moves the sprite to the front layer, making it appear in front of other sprites on the stage.

8."Go Back (1) Layers": This block moves the sprite back by a specified number of layers, placing it behind other sprites.

9."Set Size to 100%": This block sets the sprite's size to its original size (100%).

10."Set Graphic Effect to (Color)" and "Change (Color) Effect by 25": These blocks allow you to apply graphic effects like color, brightness, and fisheye distortion to the sprite.


Sound

1."Play Sound (Sound1)": This block plays the specified sound or music file, which you can upload to your Scratch project.

2."Play Drum (Xylophone)": This block plays a predefined drum sound, such as a xylophone sound.

3."Stop All Sounds": This block stops all currently playing sounds in the project.

4."Play Note (60) for (0.5) Beats": This block plays a musical note with the specified pitch (60 is middle C) for a specified duration (0.5 beats in this case).

5."Rest for (0.25) Beats": This block creates a pause in the music for the specified duration (0.25 beats in this case).

6."Change Volume by (10)": This block increases or decreases the volume of the sound by the specified amount.

7."Set Volume to (50)%": This block sets the volume of the sound to the specified percentage.

8."Change Tempo by (20)": This block increases or decreases the tempo (speed) of the music by the specified amount.

9."Set Tempo to (120) bpm": This block sets the tempo of the music to the specified beats per minute (bpm).

10."When (Sound1) Finished": This block triggers an event when the specified sound has finished playing.

Event

1."When Green Flag Clicked": This block is the starting point of your program. It runs all the code inside it when the green flag at the top of the Scratch stage is clicked.

2."When (Sprite1) Clicked": This block triggers an event when the specified sprite is clicked.

3."When (Key Space) Pressed": This block responds to a key press event. In this example, it triggers when the spacebar is pressed.

4."When This Sprite Clicked": This block responds to a click event specifically on the sprite that contains the code.

5."When Flag Clicked": Similar to "When Green Flag Clicked," this block starts the program when the green flag is clicked, but it can be used inside a sprite to respond to multiple start points.

6."When (Sprite1) Bumped": This block triggers an event when the specified sprite collides with another sprite.

7."When (Sprite1) > (10)": This block runs the code inside it when the specified condition is met. For example, it can be used to trigger an event when the x-coordinate of "Sprite1" is greater than 10.

8."When Timer > (5) secs": This block responds to a timer event. It triggers when the specified timer reaches a certain value (in this case, 5 seconds).

9."Broadcast (message1)": This block broadcasts a message to other sprites, allowing you to communicate and coordinate actions between different parts of your project.

Operators

1."Addition (+)": This block adds two numbers together.

2."Subtraction (-)": This block subtracts one number from another.

3."Multiplication (*)": This block multiplies two numbers.

4."Division (/)": This block divides one number by another.

5."Modulo (mod)": This block calculates the remainder after division.

6."Random (1) to (10)": This block generates a random number between the specified range (1 and 10 in this case).

7."Join (Hello) (World)": This block concatenates two strings, joining them together.

8."Letter (1) of (Hello)": This block extracts a specific letter from a string.

9."Not": This block negates a Boolean value (true becomes false, and false becomes true).

10."Equals (=)": This block checks if two values are equal.

11."Greater than (>)" and "Less than (<)": These blocks compare two numbers or strings.

12."And" and "Or": These blocks perform logical operations. "And" returns true if both conditions are true, and "Or" returns true if either condition is true.

13."Join (text) and (text)": This block allows you to concatenate two pieces of text.

14."Report (item1) + (item2)": This block concatenates two items together, such as strings or numbers.

Variable

1."Set (variable) to (value)": This block sets the value of a variable to a specific number or text.

2."Change (variable) by (value)": This block adds or subtracts a value from the current value of a variable.

3."Show Variable (variable)": This block displays the value of a variable on the screen.

4."Hide Variable (variable)": This block hides the variable's value from being displayed on the screen.

5."Variable (variable)": This block holds the value of a variable, which can be used in calculations or as part of other blocks.

6."Ask () and wait": This block allows you to ask the user a question and store their response in a variable.

7."Join (text) (variable)": This block concatenates a piece of text with the value of a variable.

8."Item () of (list)": This block retrieves a specific item from a list and can store it in a variable.

9."Delete () of (list)": This block removes an item from a list.

10."Insert () at () of (list)": This block inserts an item into a list at a specific position.

Control

1."If-Then": This block checks a condition, and if it's true, the code inside the block runs.

2."If-Then-Else": This block checks a condition, and if it's true, the code inside the first part runs; otherwise, the code inside the second part (else) runs.

3."Forever": This block repeats the code inside it indefinitely, creating an infinite loop.

4."Repeat (10) times": This block repeats the code inside it a specific number of times (in this case, 10 times).

5."Wait (1) seconds": This block pauses the execution of the code for the specified number of seconds.

6."Broadcast (message1)": This block broadcasts a message to trigger events in other parts of your project.

7."When I receive (message1)": This block waits for a specific message to be broadcasted before executing its code.

8."Stop": This block stops the execution of the script immediately.

9."Wait until": This block pauses the code until a specific condition becomes true.

10."Broadcast and Wait": This block broadcasts a message and waits until other sprites have finished executing code in response to the message.

Sensing

1."Touching (Sprite2)": This block checks if the sprite is touching another sprite named "Sprite2."

2."Touching Color (Color)": This block checks if the sprite is touching a specific color on the stage.

3."Color (Color) is touching (Sprite2)": This block checks if a specific color is touching another sprite named "Sprite2."

4."Mouse X" and "Mouse Y": These blocks give you the x and y coordinates of the mouse pointer on the stage.

5."Ask (What's your name?)" and "Answer": These blocks allow your program to ask the user a question and store their response in the "Answer" variable.

6."Key (Space) pressed?": This block checks if a specific key (e.g., spacebar) is currently pressed.

7."Mouse down?": This block checks if the mouse button is currently pressed.

8."Timer": This block gives you the number of seconds since the green flag was clicked.

9."Current (Year/Month/Date/Day/Hour/Minute/Second)": These blocks provide the current date and time information.

10."Distance to (Sprite2)": This block calculates the distance between the sprite running the code and another sprite named "Sprite2."

Extension

1.Hardware Extensions: These extensions enable interaction with external hardware devices such as microcontrollers, sensors, motors, and more. Examples include extensions for LEGO Mindstorms, Makey Makey, Arduino, and Raspberry Pi.

2.Data Extensions: These extensions allow Scratch projects to access external data sources, APIs, or online services. They may provide blocks to fetch data from websites, use real-time data, or interact with online platforms.

3.Text-to-Speech and Speech Recognition: Extensions that provide the ability to convert text to speech or recognize speech inputs in a project.

4.Music and Sound Libraries: Extensions that bring additional sound effects, music, or instruments to the Scratch sound library.

5.Multiplayer and Networking: Extensions that enable multiplayer functionality, such as sharing data or interacting with other Scratch projects in real-time.
